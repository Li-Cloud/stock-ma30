**根本原因**:

- `trading_system/core/stock_scanner.py` - `_analyze_single_stock()` 方法 (第428行)
- `trading_system/core/phase_analyzer.py` - `analyze_phase()` 方法 (第218行)

**解决建议**:
1. 在访问`latest["ma30"]`之前检查ma30是否为NaN
2. 改进错误处理，记录具体的失败原因（数据不足、ma30为NaN等）

### 2. 增加重试机制 (data_collector.py)

```python
async def _request(self, endpoint: str, params: Dict[str, Any] = None, retry: int = 3) -> Optional[Dict]:
    """发送HTTP请求，支持重试"""
    import asyncio
    
    for attempt in range(retry):
        try:
            url = f"{self.base_url}{endpoint}"
            logger.debug(f"请求: {url}, 参数: {params}")
            response = await self.client.get(url, params=params)
            response.raise_for_status()
            result = response.json()
            
            # 检查返回码
            if result.get("code") != 0:
                error_msg = result.get('message', '未知错误')
                
                # 对于数据缺失错误，不重试
                if 'total=0' in error_msg or '超时' in error_msg:
                    logger.error(f"API返回错误: {error_msg}（不可重试）")
                    return None
                
                logger.warning(f"API返回错误: {error_msg}（尝试重试 {attempt+1}/{retry}）")
                if attempt < retry - 1:
                    await asyncio.sleep(1 * (attempt + 1))  # 指数退避
                    continue
                else:
                    logger.error(f"API返回错误: {error_msg}（重试失败）")
                    return None
            
            return result.get("data")
            
        except httpx.TimeoutException as e:
            logger.warning(f"请求超时 {endpoint}（尝试重试 {attempt+1}/{retry}）")
            if attempt < retry - 1:
                await asyncio.sleep(2 * (attempt + 1))
                continue
            else:
                logger.error(f"请求超时 {endpoint}（重试失败）")
                return None
                
        except httpx.ConnectError as e:
            logger.warning(f"连接失败 {endpoint}（尝试重试 {attempt+1}/{retry}）")
            if attempt < retry - 1:
                await asyncio.sleep(2 * (attempt + 1))
                continue
            else:
                logger.error(f"连接失败 {endpoint}（重试失败）")
                return None
                
        except Exception as e:
            logger.error(f"请求失败 {endpoint}: {e}")
            return None
    
    return None
```
