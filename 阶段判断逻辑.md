# 30周均线阶段判断逻辑

本文档详细说明了系统中判断股票各阶段的具体逻辑实现。

---

## 核心参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `ma_period` | 30 | 均线周期（30周） |
| `slope_threshold` | 0.02 | 斜率阈值（判断走平） |
| `consolidation_threshold` | 0.15 | 横盘阈值（15%波动范围） |
| `volume_threshold` | 2.0 | 成交量阈值（突破需2倍以上） |

---

## 各阶段判断逻辑

### 第二阶段（上升趋势）

**判断条件**：
1. 30周均线方向向上（`ma_direction == "up"`）
2. 股价在均线上方（`price_to_ma_ratio > 1.0`）
3. **或**：刚从横盘突破（最近5周内有横盘区间结束）

```python
if ma_direction == "up" and price_to_ma_ratio > 1.0:
    if consolidation and consolidation.end_idx >= len(df) - 5:
        return StockPhase.PHASE_2_RISING
    return StockPhase.PHASE_2_RISING
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `_determine_phase()` 方法

---

### 第四阶段（下降趋势）

**判断条件**：
1. 30周均线方向向下（`ma_direction == "down"`）
2. 股价在均线下方（`price_to_ma_ratio < 1.0`）

```python
if ma_direction == "down" and price_to_ma_ratio < 1.0:
    return StockPhase.PHASE_4_FALLING
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `_determine_phase()` 方法

---

### 第一阶段（底部横盘）

**判断条件**：
1. 30周均线走平（`ma_direction == "flat"`）
2. 股价在均线附近（`price_to_ma_ratio <= 1.05`）
3. 股价处于历史低位（最近100周最低价的120%范围内）

```python
if ma_direction == "flat" and price_to_ma_ratio <= 1.05:
    historical_low = df["close"].tail(100).min()
    if recent_prices[-1] < historical_low * 1.2:
        return StockPhase.PHASE_1_BOTTOM
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `_determine_phase()` 方法

---

### 第三阶段（顶部横盘）

**判断条件**：
1. 30周均线走平（`ma_direction == "flat"`）
2. 股价在均线附近（`price_to_ma_ratio >= 0.95`）
3. 股价处于历史高位（最近100周最高价的80%范围内）

```python
if ma_direction == "flat" and price_to_ma_ratio >= 0.95:
    historical_high = df["close"].tail(100).max()
    if recent_prices[-1] > historical_high * 0.8:
        return StockPhase.PHASE_3_TOP
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `_determine_phase()` 方法

---

## 默认兜底逻辑

当上述条件都不满足时，按以下规则判断：

1. **均线向上** → 第二阶段
2. **均线向下** → 第四阶段
3. **均线走平** → 根据价格与中位价的关系判断：
   - 价格 < 中位价 → 第一阶段
   - 价格 ≥ 中位价 → 第三阶段

```python
if ma_direction == "up":
    return StockPhase.PHASE_2_RISING
elif ma_direction == "down":
    return StockPhase.PHASE_4_FALLING
else:
    mid_price = (df["close"].max() + df["close"].min()) / 2
    if recent_prices[-1] < mid_price:
        return StockPhase.PHASE_1_BOTTOM
    else:
        return StockPhase.PHASE_3_TOP
```

## 辅助判断机制

### 均线方向判断

根据30周均线的斜率判断方向：

```python
def get_ma_direction(self, slope: float) -> str:
    if slope > 0.02:      # 向上
        return "up"
    elif slope < -0.02:   # 向下
        return "down"
    else:                 # 走平
        return "flat"
```

**说明**：
- 斜率 > 0.02：均线向上
- 斜率 < -0.02：均线向下
- -0.02 ≤ 斜率 ≤ 0.02：均线走平

**代码位置**：`trading_system/core/phase_analyzer.py` - `get_ma_direction()` 方法

---

### 横盘区间检测

**检测条件**：
- 价格波动范围 < 15%（相对于平均价格）
- 30周均线标准差 < 0.02（相对均值）
- 最小持续时间：8周

```python
price_range = (window_close.max() - window_close.min()) / window_close.mean()
ma_flat = np.std(window_ma) / np.mean(window_ma) < self.slope_threshold

if price_range < self.consolidation_threshold and ma_flat:
    # 找到横盘区间
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `find_consolidation_ranges()` 方法

---

### 突破检测

**向上突破**：
- 价格 > 阻力位 × 1.03（突破3%以上）
- 成交量 > 横盘期间平均成交量 × 2

```python
resistance = consolidation.high
breakout_price = breakout_data["close"].iloc[0]
upward_breakout = breakout_price > resistance * 1.03

current_volume = breakout_data["volume"].iloc[0]
volume_confirmed = current_volume > consolidation.avg_volume * self.volume_threshold
```

**向下突破**：
- 价格 < 支撑位 × 0.97（跌破3%以上）
- 成交量 > 横盘期间平均成交量 × 2

```python
support = consolidation.low
breakdown_price = breakdown_data["close"].iloc[0]
downward_breakdown = breakdown_price < support * 0.97

current_volume = breakdown_data["volume"].iloc[0]
volume_confirmed = current_volume > consolidation.avg_volume * self.volume_threshold
```

**代码位置**：
- `trading_system/core/phase_analyzer.py` - `detect_breakout()` 方法
- `trading_system/core/phase_analyzer.py` - `detect_breakdown()` 方法

## 买卖信号验证

### 买入条件

必须同时满足以下条件：

1. **30周均线向上**：`metrics.ma30_direction == "up"`
2. **股价在均线上方**：`latest["close"] > latest["ma30"]`
3. **成交量放大**：成交量 > 2倍平均成交量
4. **突破确认**：确认有效突破横盘区间

```python
def is_valid_buy_setup(self, df: pd.DataFrame, metrics: PhaseMetrics) -> Tuple[bool, str]:
    if metrics.ma30_direction != "up":
        return False, "30周均线未向上"
    if latest["close"] <= latest["ma30"]:
        return False, "股价未站上30周均线"
    if not metrics.volume_confirmation:
        return False, "成交量未放大"
    if not metrics.breakout_confirmed:
        return False, "未确认有效突破"
    return True, "符合买入条件"
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `is_valid_buy_setup()` 方法

---

### 卖出条件

满足以下任一条件：

1. **30周均线走平或向下**：`metrics.ma30_direction != "up"`
2. **股价跌破均线**：`latest["close"] < latest["ma30"]`
3. **成交量异常放量下跌**：`metrics.volume_confirmation == True`

```python
def is_valid_sell_setup(self, df: pd.DataFrame, metrics: PhaseMetrics) -> Tuple[bool, str]:
    if metrics.ma30_direction == "up":
        return False, "30周均线仍向上，继续持有"
    if latest["close"] >= latest["ma30"]:
        return False, "股价未跌破30周均线"
    if metrics.volume_confirmation:
        return True, "放量下跌，建议卖出"
    return True, "趋势转弱，建议卖出"
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `is_valid_sell_setup()` 方法

## 阶段指标（PhaseMetrics）

每次阶段分析会返回以下指标：

| 指标 | 类型 | 说明 |
|------|------|------|
| `ma30_slope` | float | 30周均线斜率（归一化） |
| `ma30_direction` | str | 均线方向（up/down/flat） |
| `price_to_ma_ratio` | float | 股价与均线的比率 |
| `consolidation_weeks` | int | 最近横盘持续周数 |
| `breakout_confirmed` | bool | 是否确认突破 |
| `volume_confirmation` | bool | 成交量是否放大 |

## 完整分析流程

```
1. 输入：包含周线数据的DataFrame（至少150周数据）
2. 计算30周均线
3. 计算均线斜率和方向
4. 计算价格与均线关系
5. 寻找横盘区间
6. 判断阶段（按优先级）：
   - 第二阶段：均线向上 + 股价在均线上方
   - 第四阶段：均线向下 + 股价在均线下方
   - 第一阶段：均线走平 + 股价在低位
   - 第三阶段：均线走平 + 股价在高位
7. 检测突破/跌破
8. 生成阶段指标
9. 返回：阶段 + 指标
```

**代码位置**：`trading_system/core/phase_analyzer.py` - `analyze_phase()` 方法

## 理论基础

本逻辑基于史丹·温斯坦《笑傲牛熊：一条均线定乾坤》的四阶段理论：

1. **第一阶段（底部/冬天）**：30周均线走平，股价在低位震荡，观察期
2. **第二阶段（上升/春夏）**：30周均线向上，股价突破，买入并持有
3. **第三阶段（顶部/秋天）**：30周均线走平，股价在高位震荡，分批卖出
4. **第四阶段（下降/冬天）**：30周均线向下，股价下跌，远离

**核心原则**：
- 只在第二阶段买入
- 不要试图抄底摸顶
- 单笔亏损不超过总资金2-3%
- 止损位只能上移，不能下移
